課程簡介

程式設計

從零開始學習 Python，掌握變數、迴圈、函數、資料結構等核心概念

硬體組裝

認識 Crazyflie 無人機結構，完成組裝與維護，理解軟硬體整合

無人機駕駛

手動遙控飛行訓練，建立空間感與飛行安全意識

程式控制飛行

使用 Python 程式控制無人機，實現自主飛行任務

三階段學習模式

階段一：模擬器階段

W1-W8

  • Python 基礎語法學習
  • Webots 模擬器飛行控制
  • 零硬體風險，學生可在家練習

階段二：實機組裝與手控

W9-W10

  • Crazyflie 硬體組裝
  • 手動遙控訓練
  • 建立飛行安全意識

階段三：程式實飛

W11-W18

  • cflib SDK 實機控制
  • 感測器數據應用
  • 期末專題開發與展示

所需設備

軟體環境（免費）

  • Python 3.8+
  • Webots R2023b 模擬器
  • VSCode 編輯器
  • cflib Python 函式庫

硬體設備（每組約 $12,000）

  • Crazyflie 2.1 無人機 - $6,000
  • Crazyradio PA 連線模組 - $1,200
  • Flow Deck v2 定位感測器 - $1,500
  • 備用電池 x3 - $900
  • 保護罩 - $600

💡 建議 2-3 人一組共用設備
