> 🚁 透過 Crazyflie 無人機，讓高中生在有趣的實作過程中學習 Python 程式設計

---

## 📚 課程簡介

這是一個為期 **18 週**的高中 Python 程式設計課程，使用 **Crazyflie 無人機**搭配 **cflib Python SDK** 進行教學。課程整合「程式設計」、「硬體組裝」、「飛行操作」、「程式控制」四大面向，符合台灣 **108 課綱**資訊科技素養指標。

### 🎯 課程目標

完成本課程後，學生將能夠：

- ✅ 獨立撰寫 100+ 行 Python 程式
- ✅ 完成 Crazyflie 無人機組裝與維護
- ✅ 手動遙控完成基礎飛行任務
- ✅ 使用 Python 控制無人機完成自主飛行

### 📅 18 週進度總覽

| 週次 | 主題 | Python 重點 | 飛行內容 |
|:---:|------|-----------|---------|
| W1 | Python 環境建置 | 變數、資料型態、輸出 | - |
| W2 | Webots 模擬器設置 | import、物件導向 | 啟動模擬無人機 |
| W3 | 模擬起飛與降落 | 函數呼叫、time.sleep() | 起飛 → 懸停 → 降落 |
| W4 | 精準高度控制 | 函數參數、整數/浮點數 | 階梯式高度飛行 |
| W5 | 水平移動控制 | 多參數函數 | 前後左右移動 |
| W6 | 變數與運算應用 | 算術運算、變數更新 | 動態高度控制 |
| W7 | 旋轉與組合動作 | 浮點數、組合指令 | 原地旋轉 + 移動 |
| W8 | 幾何飛行挑戰 | 程式碼重構、註解 | 正方形飛行路徑 |
| **W9** | **Crazyflie 組裝** | **For 迴圈、range()** | **實機硬體組裝** |
| **W10** | **手動遙控訓練** | **While 迴圈、break** | **手控飛行認證** |
| W11 | 實機連線與首飛 | try-except、異常處理 | 第一次程式實飛 |
| W12 | 重複巡邏任務 | For 迴圈實戰 | 自動巡邏 3 次 |
| W13 | 電量檢查機制 | If-Else、布林值 | 低電量強制降落 |
| W14 | 互動式控制 | input()、字串比對 | WASD 鍵盤控制 |
| W15 | ToF 高度感測器 | 感測器讀取、回傳值 | 自動高度維持 |
| W16 | IMU 與自定義函數 | def、參數預設值 | 即時姿態顯示 |
| W17 | 航點飛行 | Lists、元組 | 自動航點飛行 |
| **W18** | **期末專題發表** | **綜合應用** | **專題展示** |

## 🚀 快速開始

### 1. 安裝軟體環境

```bash
# 安裝 Python（建議使用 Anaconda）
# 前往 https://www.anaconda.com/download

# 安裝 cflib
pip install cflib

# 安裝 Webots
# 前往 https://cyberbotics.com/download
```

### 2. 測試模擬器

```bash
# 下載本專案
git clone https://github.com/你的帳號/Python-Flight-Academy.git
cd Python-Flight-Academy

# 開啟 Webots 並載入 Crazyflie 範例
# File > Open World > crazyflie.wbt
```

### 3. 執行第一個程式

```python
from controller import Robot

drone = Robot()
print("✈️ 我的第一台模擬無人機！")
```

## 🏆 期末專題範例

學生可自訂題目，以下是範例：

1. **自動障礙接力賽** - 通過三個高度不同的門框
