#### 2-2. 變數與資料型態（20 分鐘）

**概念引入**：
```
變數就像是一個「盒子」，用來裝資料
盒子有「名字」（變數名稱）
盒子裡的東西（變數值）可以改變
```

**基本資料型態**：

**1. 整數（Integer, int）**
```python
age = 16
student_count = 30
year = 2026

print(age)           # 輸出：16
print(type(age))     # 輸出：<class 'int'>
```

**2. 浮點數（Float）**
```python
height = 1.75
weight = 62.5
pi = 3.14159

print(height)        # 輸出：1.75
print(type(height))  # 輸出：<class 'float'>
```

**3. 字串（String, str）**
```python
name = "王小明"
school = "內湖高工"
message = '歡迎來到 Python 的世界！'

print(name)          # 輸出：王小明
print(type(name))    # 輸出：<class 'str'>
```

**4. 布林值（Boolean, bool）**
```python
is_student = True
has_license = False

print(is_student)    # 輸出：True
print(type(is_student))  # 輸出：<class 'bool'>
```

**變數命名規則**：
```python
# ✅ 正確的命名
my_age = 16
student_name = "小明"
total_score = 95

# ❌ 錯誤的命名
2nd_name = "錯誤"      # 不能以數字開頭
my-age = 16           # 不能使用連字號
class = "A"           # 不能使用保留字
```

**教學重點**：
- 變數名稱要有意義（避免 a, b, c）
- 使用小寫字母和底線（snake_case）
- 不能使用 Python 保留字

**學生練習**（10 分鐘）：
```python
# 練習：建立變數描述自己
name = "_______"       # 你的名字
age = _______          # 你的年齡
height = _______       # 你的身高（公尺）
is_male = _______      # 是否為男生（True/False）

# 輸出結果
print("姓名：", name)
print("年齡：", age)
print("身高：", height, "公尺")
```

#### 2-3. 字串格式化（15 分鐘）

**基本輸出**：
```python
name = "小明"
age = 16

# 方法 1：使用逗號分隔
print("我是", name, "，今年", age, "歲")
# 輸出：我是 小明 ，今年 16 歲
```

**格式化字串（f-string）**：
```python
name = "小明"
age = 16

# 方法 2：使用 f-string（推薦！）
print(f"我是 {name}，今年 {age} 歲")
# 輸出：我是 小明，今年 16 歲

# 可以在 {} 中進行運算
print(f"明年我就 {age + 1} 歲了")
# 輸出：明年我就 17 歲了
```

**實用範例**：
```python
# 無人機參數設定
height = 0.5
speed = 0.3

print(f"無人機高度：{height} 公尺")
print(f"飛行速度：{speed} m/s")
print(f"預估到達時間：{10 / speed:.2f} 秒")
# 輸出：預估到達時間：33.33 秒
```

**格式化選項**：
```python
pi = 3.14159265359

print(f"{pi}")        # 輸出：3.14159265359
print(f"{pi:.2f}")    # 輸出：3.14（小數點後 2 位）
print(f"{pi:.4f}")    # 輸出：3.1416（小數點後 4 位）
```

**學生練習**（5 分鐘）：
```python
# 練習：計算 BMI
name = "______"
height = _______  # 單位：公尺
weight = _______  # 單位：公斤

bmi = weight / (height ** 2)

print(f"{name} 的 BMI 是 {bmi:.2f}")
```

#### 2-4. 基礎運算（5 分鐘）

**算術運算子**：
```python
# 加減乘除
a = 10
b = 3

print(f"{a} + {b} = {a + b}")   # 加法：13
print(f"{a} - {b} = {a - b}")   # 減法：7
print(f"{a} * {b} = {a * b}")   # 乘法：30
print(f"{a} / {b} = {a / b}")   # 除法：3.333...

# 特殊運算
print(f"{a} // {b} = {a // b}") # 整數除法：3
print(f"{a} % {b} = {a % b}")   # 取餘數：1
print(f"{a} ** {b} = {a ** b}") # 次方：1000
```

**無人機應用範例**：
```python
# 計算飛行距離
speed = 0.5      # 速度：0.5 m/s
time = 10        # 時間：10 秒
distance = speed * time

print(f"飛行距離：{distance} 公尺")

# 計算飛行時間
target_distance = 20  # 目標距離：20 公尺
flight_time = target_distance / speed

print(f"需要飛行 {flight_time} 秒")
```
